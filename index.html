DSA Roadmap Checklist

Phase 1: Fundamentals & Analysis ▶
Time Complexity & Big-O Notation
Arrays: insertion, deletion, traversal, max/min, prefix sum
HashMaps / Sets: frequency, duplicates, two-sum
Phase 2: Patterns on Arrays/Strings ▶
Sliding Window
Two Pointers
Fast & Slow Pointer
Modified Binary Search
Phase 3: Linked Lists ▶
Single & Doubly Linked Lists
Reverse Linked List
Detect Cycle
Find Middle Node
Phase 4: Recursion & Backtracking ▶
Recursion Basics
Backtracking: subsets, permutations, combinations
Phase 5: Stacks & Queues ▶
Stack: reverse string, balanced parentheses
Queue: Enqueue/Dequeue, BFS usage
Phase 6: Trees ▶
Binary Trees / BST
BFS (level order), DFS (pre/in/postorder)
Path sum, height, diameter
Phase 7: Graphs ▶
Graph Intro: adjacency list/matrix
Graph Traversals: BFS & DFS
Advanced: Topological Sort, Shortest Path
Phase 8: Tries & Priority Queue ▶
Trie: word search, prefix problems
Priority Queue / Heap: Top K, K-way merge, median
Phase 9: Sorting ▶
Basic: Bubble, Selection, Insertion
Advanced: Merge Sort, Quick Sort
Phase 10: Dynamic Programming ▶
Simple DP: Fibonacci, Climbing Stairs
Advanced DP: 0/1 Knapsack, Subset Sum, LIS
Phase 11: Extra Patterns ▶
Merge Intervals
Cyclic Sort
Two Heaps
K-Way Merge
